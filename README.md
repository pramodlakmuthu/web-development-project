# 🩸 Red Drop - Donate blood, Save Lifes

## 📋 About the Project

**Red Drop** is a modern, user-friendly web application dedicated to promoting blood donation and connecting donors with those in need. Our mission is to make blood donation accessible and encourage people to become heroes by saving lives through simple acts of kindness.

## 🎯 Vision

We believe that one drop of blood can save up to three lives. Red Drop is built to remove barriers to blood donation and create a community of compassionate donors who understand the life-saving impact of their contribution.

## ✨ Key Features

- ✅ **Eligibility Checker**: A quick and interactive tool to help potential donors determine if they're eligible to donate blood based on health guidelines
- 📚 **Educational Content**: Comprehensive information about why blood donation matters and who needs it
- 💪 **Health Benefits**: Highlights the personal health benefits donors receive, including free health check-ups
- 🤝 **Community Impact**: Emphasizes the role of donors in maintaining safe blood supplies in hospitals and blood banks
- 📱 **Responsive Design**: Fully responsive interface that works seamlessly on desktop, tablet, and mobile devices

## 🏥 Who Benefits?

Blood donations help:

- 🚨 Accident and trauma victims
- 🎗️ Cancer patients requiring transfusions
- 👩‍🍼 Pregnant mothers and postpartum women
- 🏨 Surgical patients
- 💊 Patients with chronic diseases

## 🚀 Getting Started

### 📌 Prerequisites

- 🔧 Node.js installed on your system
- 📦 npm (comes with Node.js)

### 💻 Installation

1. Navigate to the project directory
2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

The application will automatically open in your default web browser.

## 🛠️ Technologies Used

- 🎨 **HTML5**: Semantic markup for the web interface
- 💎 **Tailwind CSS**: Utility-first CSS framework for responsive design
- 🎭 **Font Awesome**: Icon library for visual elements
- 🟢 **Node.js**: Backend server environment
- 🌐 **http-server**: Lightweight HTTP server for development
